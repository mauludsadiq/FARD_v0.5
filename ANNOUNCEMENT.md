FARD 1.0.0 — RELEASE ANNOUNCEMENT
=====================================
Date: 2026-03-10

FARD is a pure, deterministic, content-addressed scripting language.
Every program run produces a cryptographic digest committing to
all inputs, outputs, and intermediate computation steps.

WHAT'S IN v1.0.0
----------------
  - 22 stdlib modules, 144+ functions
  - Arbitrary precision integers (std/bigint)
  - Full regex support (std/re)
  - HTTP client (std/http)
  - Cryptographic hash chain on every computation (while loop)
  - Content-addressed digest on every program run
  - Channels and mutexes (std/chan, std/mutex)
  - Async/await via promise primitive (std/promise)
  - Early return from functions
  - Named/keyword arguments
  - Multiline strings (backtick syntax)
  - AST access and macro system (std/ast + std/eval)
  - Pattern matching with match/type dispatch
  - Bigint, UUID, datetime, CSV, base64, path, JSON
  - Module system with import from .fard files
  - Self-verifying test suite: 250+ tests in pure FARD

PHILOSOPHY
----------
FARD programs are not just code — they are certified computations.
The while loop is not mutable iteration; it is a hash-chained
ledger of every state transition. Every run is signed.
